# TabulaKit for VS Code

Companion extension for [TabulaKit](https://github.com/heatherstoneio/tabulakit) documentation sites.

## Features

- **Live Preview** — Embedded site preview with auto-start local server
- **Dashboard** — Three-tab interface: Site Preview, TabulaKit controls, and Help
- **Auto-detection** — Activates automatically when you open a TabulaKit project
- **Per-site ports** — Each TabulaKit site gets its own preview port (no conflicts)
- **Update checking** — Check for upstream template updates

## Getting Started

1. Install this extension
2. Open a TabulaKit project folder in VS Code
3. The TabulaKit icon appears in the activity bar — click it to open the dashboard

## What is TabulaKit?

TabulaKit is a template for creating documentation sites with zero build step. Fork the repo, run `/startup` in Claude Code, and deploy in 5 minutes.

Learn more at [heatherstoneio.github.io/tabulakit](https://heatherstoneio.github.io/tabulakit/).

## Commands

| Command | Description |
|---------|-------------|
| `TabulaKit: Show Dashboard` | Open the TabulaKit dashboard |
| `TabulaKit: Refresh Preview` | Reload the site preview |
| `TabulaKit: Reset Layout` | Reset VS Code layout for TabulaKit |
| `TabulaKit: Check for Updates` | Check for template updates |
